# Data Analysis Project

University of Basel · Fall 2025 · Database Systems

The choice between public transport or a personal vehicle for the daily commute is commonly based on personal preference rather than objective reasoning. We wonder if it might be possible to instead base this decision on hard data: specifically, we wonder if it's possible to measure a difference in reliability between the two options in the face of suboptimal weather conditions.
